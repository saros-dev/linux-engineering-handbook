# Linux History

> “Linux is not just an operating system — it is the result of decades of evolution in operating systems, academic research, and open-source collaboration.”

---

## 🧭 Overview

The history of Linux is deeply rooted in the evolution of Unix systems in the 1970s and the open-source movement of the 1990s.

Linux did NOT appear suddenly.

It is the result of:

- Unix (1970s)
- MINIX (educational OS)
- GNU Project (free software movement)
- Linus Torvalds' kernel (1991)

---

## ⏳ Timeline of Linux Evolution

```text id="timeline1"
1970s  → Unix developed at Bell Labs (Ken Thompson, Dennis Ritchie)
1980s  → MINIX created for education (Andrew Tanenbaum)
1983   → GNU Project started (Richard Stallman)
1991   → Linux kernel released by Linus Torvalds
1992   → Linux becomes open source (GPL license)
1993+  → Distributions (Debian, Slackware, Red Hat)
Today  → Linux powers servers, cloud, Android, supercomputers
```

---

## 🧬 The Unix Foundation

Unix was created at **Bell Labs** and became the foundation of modern operating systems.

Key contributors:

- Ken Thompson
- Dennis Ritchie

Unix introduced:

- Hierarchical file system
- Everything is a file concept
- Process-based execution
- Multi-user systems

---

## 🧪 MINIX (The Missing Link)

MINIX was an educational OS created by Andrew S. Tanenbaum.

It was:

- Lightweight Unix-like system
- Used for teaching OS concepts
- NOT production ready

Linus Torvalds used MINIX as inspiration but wanted something better.

---

## 🐧 The Birth of Linux (1991)

In 1991, Linus Torvalds posted this famous message:

> “I'm doing a (free) operating system (just a hobby, won't be big and professional like GNU)...”

He released Linux kernel v0.01.

It included:

- Basic process management
- File system support
- Terminal interface
- Basic drivers

---

## 📜 Open Source Revolution

Linux became powerful because of the **GNU General Public License (GPL)**.

This meant:

- Anyone can use Linux
- Anyone can modify it
- Anyone can distribute it

This triggered the global open-source movement.

---

## 🧱 Linux vs GNU

```text id="gnu1"
GNU Project → User space tools (bash, gcc, coreutils)
Linux Kernel → Core system (CPU, memory, devices)

Together = GNU/Linux system
```
